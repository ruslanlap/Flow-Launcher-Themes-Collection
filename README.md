Welcome to the comprehensive collection of Flow Launcher themes! This repository automatically aggregates and organizes theme submissions from the [Flow Launcher Theme Gallery discussion](https://github.com/Flow-Launcher/Flow.Launcher/discussions/1438).

## 🚀 Quick Start

1. **Browse** themes by category or use the search filters
2. **Download** your favorite theme
3. **Install** by placing the `.xaml` file in your Flow Launcher themes directory
4. **Activate** in Flow Launcher settings
